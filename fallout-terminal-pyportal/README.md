
# Fallout Terminal — Home Assistant Lighting Controller

A wall-mounted lighting controller styled as a RobCo Industries terminal from the Fallout universe. Built on an [Adafruit PyPortal](https://www.adafruit.com/product/4116), it displays a phosphor-green terminal interface on a 3.2" touchscreen and publishes sensor readings and light commands to Home Assistant via MQTT.

## What it does

- Displays time, temperature, humidity, and ambient light on a main dashboard screen
- Touch the screen to open a full lighting control panel with per-zone on/off toggles and brightness controls
- Publishes temperature and light sensor readings to Home Assistant as auto-discovered MQTT sensors
- Plays Fallout-style sound effects on interaction
